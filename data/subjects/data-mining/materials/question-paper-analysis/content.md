# **Module 1: Introduction to Data Mining and Data Warehousing**

**🎯 Exam Strategy & Planning**

Part B consistently offers a choice between Data Warehousing concepts (Architecture/OLAP) and Data Mining foundations (KDD/Issues). Preparing thoroughly for **Data Warehousing architectures and OLAP operations** is the safest bet for securing a full 14-mark question.

**🔴 High Priority (Repeated 3+ Times)**

- **OLAP Operations:** Explain basic operations with examples.
- **Data Warehouse Architecture:** Detailed explanation with diagrams (including 3-tier).
- **Knowledge Discovery in Databases (KDD):** Explain the stages/process with a neat diagram.
- **Data Mining Issues:** Discuss various issues in data mining (often asked for 3-6 marks).

**🟡 Medium Priority (Repeated 2 Times)**

- **Multidimensional Data Model:** Explain with examples.
- **OLTP vs OLAP:** Differentiate between the two systems.
- **Data Mining Functionalities:** Summarize functionalities.

**🟢 Low Priority (Appeared Once)**

- Warehouse Schema definitions and types (Star vs Snowflake).
- Data Mining vs Data Warehousing differentiation.

# **Module 2: Data Preprocessing**

**🎯 Exam Strategy & Planning**

Part B almost guarantees one numerical problem based on statistical analysis (Chi-Square or Linear Regression). The "OR" option usually covers dimensionality reduction (PCA/DWT) or data sampling theory. Mastering the **Chi-Square test and Linear Regression formulas** is the most predictable path.

**🔴 High Priority (Repeated 3+ Times)**

- **Chi-Square Test of Independence:** Determine significant association/independence between two attributes using a contingency table.
- **Linear Regression:** Develop a model to predict values (e.g., sales, fuel efficiency) based on a given dataset.
- **Data Smoothing Techniques:** Apply binning methods (bin boundaries, bin means) to a given array.

**🟡 Medium Priority (Repeated 2 Times)**

- **Data Normalization:** Apply Min-Max normalization and Z-score normalization to a dataset.
- **Principal Component Analysis (PCA):** Apply dimension reduction/feature dimensionality reduction on a dataset.
- **Discrete Wavelet Transform (DWT):** Calculate coefficients for a given array.

**🟢 Low Priority (Appeared Once)**

- Attribute subset selection procedures (forward, backward).
- Sampling techniques (SRSWOR, SRSWR, Stratified).
- Methods to handle missing data.

# **Module 3: Advanced classification and Cluster analysis**

**🎯 Exam Strategy & Planning**

The structure of Part B in this module is highly predictable. One full question will test **Classification** (building an ID3 Decision Tree and calculating Gini Index), while the "OR" question tests **Clustering** (tracing K-Means or K-Medoids iterations). Choose one domain to master perfectly. Part A will definitely contain a precision/recall calculation.

**🔴 High Priority (Repeated 3+ Times)**

- **Decision Tree Construction (ID3 Algorithm):** Calculate entropy and information gain to select the splitting attribute and build the tree.
- **K-Means Clustering:** Apply the algorithm with Euclidean distance to find clusters after specific epochs/iterations given initial centroids.
- **Gini Index Calculation:** Compute the overall Gini index and determine the first split attribute.
- **Classification Accuracy Measures:** Calculate Precision, Recall, and F1-score from a given true positive/false positive scenario.

**🟡 Medium Priority (Repeated 2 Times)**

- **Hierarchical Clustering:** Compare agglomerative with divisive clustering.

**🟢 Low Priority (Appeared Once)**

- K-Medoids (PAM) clustering algorithm trace.
- DBSCAN algorithm explanation.
- Distance metric calculations (Manhattan, Euclidean).

# **Module 4: Association Rule Analysis**

**🎯 Exam Strategy & Planning**

This is the most straightforward module to plan for. The Part B structure is virtually fixed: **Question 1 is always the Apriori Algorithm**, and the **"OR" Question 2 is always the FP-Growth Algorithm**. You only need to learn one of these algorithms deeply to guarantee 14 marks.

**🔴 High Priority (Repeated 3+ Times)**

- **Apriori Algorithm (14 Marks):** Find frequent itemsets and generate association rules based on given support and confidence thresholds.
- **FP-Growth Algorithm (14 Marks):** Construct an FP-tree, extract frequent itemsets using conditional pattern bases, and generate rules.
- **Support and Confidence Basics (3 Marks):** Calculate support and confidence values from a small sample dataset.

**🟡 Medium Priority (Repeated 2 Times)**

- No medium priority topics; questions are highly concentrated on the two core algorithms.

**🟢 Low Priority (Appeared Once)**

- Methods to improve Apriori efficiency.
- Real-world applications of association rules.

# **Module 5: Advanced Data Mining Techniques**

**🎯 Exam Strategy & Planning**

Part B typically divides into **Web Mining** and **Text Mining**. Questions are largely descriptive and theoretical. Preparing the core components of Text Mining (Indexing and Retrieval) and the PageRank algorithm provides the best coverage.

**🔴 High Priority (Repeated 3+ Times)**

- **Web Content Mining:** Explain the concept and list/describe 3-4 techniques used.
- **Text Indexing & Retrieval:** Explain indexing techniques, retrieval methods, and query processing in text mining.
- **Basic Measures for Text Retrieval (3 Marks):** Describe the fundamental metrics used.

**🟡 Medium Priority (Repeated 2 Times)**

- **PageRank Algorithm:** Explain the structure mining algorithm.
- **Web Usage Mining Preprocessing:** Explain the preprocessing steps and data structures used.
- **Text Data Analysis:** Describe how it is carried out in text mining.

**🟢 Low Priority (Appeared Once)**

- Compare web content mining with web structure mining.
- Applications of web structure mining.

# **Part A: 3-Mark Short Answer Questions**

**🎯 Exam Strategy & Planning**
Part A is highly predictable and unusually calculation-heavy for a short-answer section. Securing full marks relies on mastering quick, repetitive statistical formulas. You will almost certainly face a 3-mark question on Precision/Recall/F1-score, a Support/Confidence calculation, and a Data Smoothing/Normalization array problem. The theoretical questions are brief and heavily recycle core definitions.

**🔴 High Priority (Repeated 3+ Times)**

- **Classification Accuracy Measures:** Calculate Precision, Recall, and F1-score from given true/false positive scenarios, such as spam detection or medical diagnostics.
- **Association Rule Metrics:** Calculate Support and Confidence values for specific itemsets or rules using a provided transaction table.
- **Data Smoothing:** Apply techniques like bin boundaries and bin means to smooth a provided array of numbers.
- **Text Retrieval:** Describe the basic measures used for text retrieval.

**🟡 Medium Priority (Repeated 2 Times)**

- **Data Mining Basics:** Discuss the main issues in data mining or summarize its core functionalities.
- **Data Normalization:** Apply Min-Max normalization and Z-score normalization to a given set of values.
- **Foundational Differences:** Compare and differentiate between Data Mining and Data Warehousing concepts.
- **Web Mining:** Explain the preprocessing steps or techniques used specifically for web usage mining.

**🟢 Low Priority (Appeared Once)**

- **Module 1:** Differentiate between OLTP and OLAP systems. Compare star schema with snowflake schema.
- **Module 2:** Calculate the Chi-square value from a contingency table. Calculate the mean and covariance for a small dataset.
- **Module 3:** Compare agglomerative and divisive clustering. Extract rules from a provided visual decision tree. Calculate Manhattan and Euclidean distances between two tuples. Compute the Gini index for a basic frequency array. Explain the DBSCAN algorithm.
- **Module 4:** Outline real-world applications of association rules. Describe methods to improve the efficiency of the Apriori algorithm.
- **Module 5:** Compare web content mining with web structure mining. List applications of web structure mining. Explain the Page Rank algorithm.
